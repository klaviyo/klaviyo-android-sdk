KlaviyoMock

object KlaviyoMock

Utility class for mocking Klaviyo singleton from Java tests.

Since Klaviyo is a Kotlin object (singleton), it can only be mocked using MockK's mockkObject function, which is only available in Kotlin. This class provides a bridge for Java tests to mock and verify Klaviyo method calls.
